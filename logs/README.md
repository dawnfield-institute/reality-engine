# Logs Directory

This directory contains timestamped logs from Reality Engine runs.

## Log Format

Each run generates two files:
- `reality_engine_YYYYMMDD_HHMMSS.log` - Human-readable text log
- `reality_engine_YYYYMMDD_HHMMSS.json` - Machine-readable JSON log

## Contents

Logs capture:
- Initialization parameters
- Big Bang herniation cascade
- Evolution step metrics
- Emergence events (particles, quantum, gravity)
- Field health monitoring
- NaN/Inf detection warnings
- Conservation tracking

Files in this directory are gitignored.
